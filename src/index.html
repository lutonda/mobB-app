FAQ

Perguntas Feitas Frequentemenete

Como posso usar a API de dados monetários do MODEXX?

Você pode usar a API de dados da moeda para gerenciar melhor as necessidades de várias moedas da sua empresa. Por exemplo, muitas corporações mostram os preços em várias moedas para vender com mais eficiência a um público global. Algumas organizações integram as taxas de câmbio diretamente em seus sistemas de contabilidade para economizar tempo e gerenciar suas finanças com mais precisão. Esses são apenas alguns exemplos das muitas maneiras pelas quais nossos clientes usam os dados de moeda MODEXX.

Quais moedas eu tenho acesso?

A API de dados monetários fornece acesso a taxas para todas as moedas atuais do mundo, além de quatro metais preciosos: ouro, prata, platina e paládio. Você é bem-vindo para obter todas as taxas, mas poucas organizações realmente precisam de todas elas.
NOTA: as moedas "exóticas" não principais não são negociadas tanto. Como tal, os dados são mais difíceis de obter, menos confiáveis e, consequentemente, atualizados com menos frequência. A maioria das moedas exóticas tem uma classificação de popularidade de menos de 0,2%.

O que significa "moeda base"?

Moedas são sempre cotadas em pares. A primeira moeda é chamada de moeda base e a segunda é chamada de moeda de contrapartida. Em termos gerais, a moeda base é a moeda da qual você deseja converter e a moeda do contador é a moeda que você deseja converter.
Por exemplo, se sua empresa estiver nos Estados Unidos e seus sistemas estiverem baseados em USD, você poderá especificar o USD como sua moeda base. Isso significa que todas as taxas de câmbio nos seus resultados da API (as moedas do contador) serão cotadas em relação ao USD.

Quantas vezes posso acessar a API de dados da moeda?

O número de acessos que você recebe é baseado no pacote de serviços que você selecionou ao se inscrever.
Mini: fornece 10 solicitações de taxa de API por dia, Básico: fornece 100 solicitações de taxa de API por dia, Pro: fornece solicitações ilimitadas de taxa de API por mês.
Cada taxa retornada é considerada uma solicitação da API em relação ao seu limite de pacote mensal. Você pode solicitar várias taxas em uma única consulta.
